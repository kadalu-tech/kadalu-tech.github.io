Storage for Cloud, On-premise

Organize Volumes in multiple pools and manage the Storage Pools and Volumes using the Modern Storage Manager.

Inbuilt Monitoring

Ready to use metrics in JSON and Prometheus format.

ReST APIs and SDKs

Easily integrate Kadalu Storage with your applications easily using the ReST APIs.

Programming language bindings are available

Zero setup Web UI

Along with the rich CLI, manage Kadalu Storage from the Web UI.

Only frontend code is hosted by us, Web UI runs fully in your browser and connects to Storage Clusters that are reachable from your node. No private data collected by us.

Easily setup the Web UI within your firewall when needed
