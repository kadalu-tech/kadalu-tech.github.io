Open Source

Gluster FS project is dual licensed under GPLv2 and LGPLv3.

Easy to use

Gluster FS is very easy to use and get started with it in minutes.

No metadata server

Gluster FS doesn't need a seperate server to store file system metadata. Instead, it stores both data and metadata in respective files or directories.

POSIX compatible

Gluster FS provides a POSIX compliant file system that helps your applications to use the file system without changing.

No vendor lockin

Works with any filesystem backends (XFS, ZFS, EXT4 ...) that supports extended attributes.

Multiple access protocols

Easily access the data using Fuse, NFS, SMB or GFAPI access protocols.

Volume Types

Gluster FS supports multiple volume types for different use cases.
