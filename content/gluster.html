GlusterFS project maintenance by Kadalu Technologies

The GlusterFS is an Open Source project funded by Red Hat. The project activities are slowed down and the project is not in healthy condition. The GlusterFS project needs support from the community and the companies. (Ref: Red Hat announced End of Life for their Gluster based product and it is supported till Dec 2024)

Kadalu Technologies started at the end of the year 2019. This company was started by a few GlusterFS core team members to create an Opinionated storage solution based on GlusterFS. We developed many new projects to improve the GlusterFS experience and monitoring. Kadalu Kubernetes Storage is the most popular project from Kadalu Technologies that provides storage for the apps running in Kubernetes. Kadalu Storage is an opinionated storage solution based on GlusterFS.
